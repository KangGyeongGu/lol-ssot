# BE SPEC Rule Index

## 목적
- `06_BACKEND/SPEC/`에 정의된 백엔드 명세 룰을 빠르게 탐색하기 위한 인덱스다.

## 파일 매핑
- `BE_ARCHITECTURE.md`: 아키텍처/레이어/의존성 경계
- `BE_API_RULES.md`: REST 계약 및 API 구현 규칙
- `BE_DATA_MODEL_RULES.md`: 엔티티/스키마/영속성 규칙
- `BE_REALTIME_RULES.md`: 실시간(STOMP/이벤트) 계약 규칙

## 운영 원칙
- 신규 BE-S-* 룰을 추가할 때 해당 룰의 정의 파일과 함께 이 인덱스를 갱신한다.
