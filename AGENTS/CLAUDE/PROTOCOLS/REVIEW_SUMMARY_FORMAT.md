# Review Summary Format

MASTER_PLAN.md should include:
1. Execution log
2. SSOT mismatch summary
3. QUALITY issue summary
4. Deduplication decisions
5. Consolidated task list
6. Assumptions and risks

User-facing Korean summary should include:
1. SSOT/QUALITY execution log
2. SSOT mismatch summary by severity
3. QUALITY issue summary by severity
4. Deduplication result
5. Proposed tasks
6. Report files used
7. Approval request
